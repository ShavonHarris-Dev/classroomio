# Cloudflare Setup

In this guide, you will learn how to get your cloudflare environment variables. We use Cloudflare R2 for video uploads and the aim of this guide is to help you get the right variables to properly setup our backend.

## How to get your Cloudfare Environment Variables 

## Variable Example
```bash
#not-required
CLOUDFLARE_BUCKET_DOMAIN=HTTPS://CDN.YOURDOMAIN.COM

#required
CLOUDFLARE_ACCOUNT_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
CLOUDFLARE_ACCESS_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
CLOUDFLARE_SECRET_ACCESS_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
CLOUDFLARE_BUCKET_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx


```
